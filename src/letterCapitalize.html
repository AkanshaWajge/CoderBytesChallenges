Capitalizing certain characters

For this challenge you will be capitalizing certain characters in a string.

Have the function LetterCapitalize(str) take the str parameter being passed and capitalize the first letter of each word. Words will be separated by only one space.

                    function titleCase(str) {

                        var splitStr = str.toLowerCase().split(' ');

                        for (var i = 0; i < splitStr.length; i++) {

                            // You do not need to check if i is larger than splitStr length, as your for does that for you
                            // Assign it back to the array
                            //Syntax of substring : string.substring(Startindex, Endindex)

                            splitStr[i] = splitStr[i].charAt(0).toUpperCase() + splitStr[i].substring(1);     
                        }

                        // Directly return the joined string syntax : array.join(separator)
                        
                        return splitStr.join(' '); 
                        }
